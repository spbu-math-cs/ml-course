Задача 2¶

$A \in \mathbb{R}^{n\times n}$. $\nabla_x x^TAx$.

Решение¶

$ \frac{\partial}{\partial x_i} x^TAx = \frac{\partial}{\partial x_i}\sum\limits_j x_j (Ax)_j = \frac{\partial}{\partial x_i}\sum\limits_j x_j \bigg(\sum\limits_k a_{jk}x_k\bigg) = \\ = \frac{\partial}{\partial x_i}\sum\limits_{j,k} a_{jk} x_j x_k = \\ = \sum\limits_{j \neq i} a_{ji} x_j + \sum\limits_{k \neq i} a_{ik} x_k + 2a_{ii}x_i = \sum\limits_{j} a_{ji} x_j + \sum\limits_{k} a_{ik} x_k = \sum\limits_{j} (a_{ji} + a_{ij}) x_j$

$\nabla_x x^TAx = (A + A^T)x$

Задача 3¶

$A \in \mathbb{R}^{n\times n}$. $\nabla_A \det A$.

Решение¶

:

$\frac{\partial}{\partial a_{ij}} \det A = \frac{\partial}{\partial a_{ij}}\bigg[\sum\limits_k (-1)^{i+k}a_{ik}M_{ik}\bigg] = (-1)^{i+j}M_{ij}, \; $ $M_{ik}$ $A$.

$(A^{-1})_{ij} = \frac{1}{\det A}(-1)^{i+j}M_{ji}.$

, $\nabla_A \det A = (\det A) A^{-T}$.

Задача 4¶

$A \in \mathbb{R}^{n \times n},\ B \in \mathbb{R}^{n \times n}$. $\nabla_A \text{tr}(AB)$.

Решение¶

$ \frac{\partial}{\partial a_{ij}} \text{tr}(AB) = \frac{\partial}{\partial a_{ij}} \sum\limits_k (AB)_{kk} = \frac{\partial}{\partial a_{ij}} \sum\limits_{k,l} a_{kl}b_{lk} = b_{ji}. $

$ \text{ }\nabla_A \text{tr}(AB) = B^T.$
